
## 🔐 Authentication


**Better Auth** integration supporting:

- **OAuth Providers**: Google, GitHub, LinkedIn.
- **Email/Password**: Secure credentials flow.
- **Account Linking**: Connect multiple providers to one account.

## 💳 Payments


**Dodo Payments** integration for:

- **Subscriptions**: Recurring revenue made easy.
- **One-time Payments**: For lifetime deals or credits.
- **Webhooks**: Automatically handle payment events.

## 📝 Content Management (CMS)


**Notion** as a CMS:

- Edit your landing page, blog, and documentation directly in Notion.
- Instant updates without redeploying code.
- structured databases for Hero, Features, Testimonials, and more.

## 📧 Email


**Resend** integration:

- Beautiful, React-based email templates.
- Transactional emails (Welcome, Password Reset).
- Marketing capabilities.

## 🗄️ Database & Storage

- **PostgreSQL**: Robust relational database.
- **Prisma ORM**: Type-safe database access.
- **Redis (Upstash)**: High-performance caching and rate limiting.
- **Vercel Blob**: Cloud storage for user uploads (avatars, files).

## 📊 Observability

- **BetterStack**: Uptime monitoring and log management.
- **Google Analytics**: Integrated visitor tracking.
- **OpenTelemetry**: Ready for advanced tracing.
