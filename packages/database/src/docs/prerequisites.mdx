
Before you begin, ensure your development environment meets the following requirements.


## System Requirements

- **Operating System**: macOS, Windows (WSL recommended), or Linux.
- **Node.js**: Version 20 or higher is **required**.
    - Check version: `node -v`
- **Package Manager**: **pnpm** (preferred), npm, or yarn.
    - Install pnpm: `npm install -g pnpm`

## Accounts & Services


You will need accounts with the following services to fully utilize the boilerplate features. Free tiers are sufficient for development.

1. [**GitHub**](https://github.com/): For version control and OAuth.
2. [**Vercel**](https://vercel.com/): For deployment and Blob storage.
3. [**Notion**](https://www.notion.so/): For managing website content.
4. [**Neon**](https://neon.tech/) / [**Supabase**](https://supabase.com/): For a hosted PostgreSQL database (or use local Docker).
5. [**Upstash**](https://upstash.com/): For Redis.
6. [**Resend**](https://resend.com/): For sending emails.
7. [**Dodo Payments**](https://dodopayments.com/): For payment processing.
8. [**Google Cloud Console**](https://console.cloud.google.com/): For Google OAuth.
9. [**LinkedIn Developers**](https://www.linkedin.com/developers/): For LinkedIn OAuth.
