
## "Batteries Included"


We believe you shouldn't have to reinvent the wheel. Features like authentication, emails, and payments are commodities. This boilerplate provides them out of the box so you can focus on what makes your product unique: your business logic.


## Opinionated but Flexible


We make choices for you (Prisma over Drizle, Tailwind over CSS modules) to reduce decision fatigue. However, the code is entirely yours. There are no hidden vendor lock-ins or "magic" wrappers that you can't eject from.


## The Monorepo Advantage


We use **TurboRepo** to manage a monorepo structure. This allows you to:

- Share code (UI, Types) between different apps (e.g., Web App, Marketing Site, Admin Dashboard).
- Cache builds for faster CI/CD.
- Keep your entire project context in one place.

## Design First


Aesthetics matter. We include a beautiful, responsive UI kit based on **shadcn/ui**, ensuring your app looks professional from day one.
